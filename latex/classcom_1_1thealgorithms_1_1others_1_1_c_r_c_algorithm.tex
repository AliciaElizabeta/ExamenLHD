If check == true, the divided\+Messaage is examined, in order to see if it contains any 1\textquotesingle{}s. If it does, the message is considered to be wrong by the receiver,so the variable wrong\+Mess\+Caught changes. If it does not, it is accepted, so one of the variables correct\+Mess, wrong\+Mess\+Not\+Caught, changes. If check == false, the diviided Message is added at the end of the Array\+List$<$integer$>$ message.

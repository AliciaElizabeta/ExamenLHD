\doxysubsection{Detailed Description}
Given a string, moving several characters in front of the string to the end of the string. For example, move the two characters\textquotesingle{}a\textquotesingle{} and \textquotesingle{}b\textquotesingle{} in front of the string \char`\"{}abcdef\char`\"{} to the end of the string, so that the original string becomes the string \char`\"{}cdefab\char`\"{}

\doxysubsection{Detailed Description}
This class implements a \mbox{\hyperlink{classcom_1_1thealgorithms_1_1datastructures_1_1queues_1_1_generic_array_list_queue}{Generic\+Array\+List\+Queue}}.

A \mbox{\hyperlink{classcom_1_1thealgorithms_1_1datastructures_1_1queues_1_1_generic_array_list_queue}{Generic\+Array\+List\+Queue}} data structure functions the same as any specific-\/typed queue. The \mbox{\hyperlink{classcom_1_1thealgorithms_1_1datastructures_1_1queues_1_1_generic_array_list_queue}{Generic\+Array\+List\+Queue}} holds elements of types to-\/be-\/specified at runtime. The elements that are added first are the first to be removed (FIFO). New elements are added to the back/rear of the queue.

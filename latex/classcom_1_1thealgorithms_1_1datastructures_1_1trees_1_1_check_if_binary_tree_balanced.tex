\doxysubsection{Detailed Description}
This class will check if a \mbox{\hyperlink{classcom_1_1thealgorithms_1_1datastructures_1_1trees_1_1_binary_tree}{Binary\+Tree}} is balanced. A balanced binary tree is defined as a binary tree where the differenced in height between the left and right subtree of each node differs by at most one.

This can be done in both an iterative and recursive fashion. Below, {\ttfamily \mbox{\hyperlink{classcom_1_1thealgorithms_1_1datastructures_1_1trees_1_1_check_if_binary_tree_balanced_a8f768e75190f9d8e39eef48b71037389}{is\+Balanced\+Recursive()}}} is implemented in a recursive fashion, and {\ttfamily \mbox{\hyperlink{classcom_1_1thealgorithms_1_1datastructures_1_1trees_1_1_check_if_binary_tree_balanced_a7f2b9b030c04aba995064b2daefe641c}{is\+Balanced\+Iterative()}}} is implemented in an iterative fashion.

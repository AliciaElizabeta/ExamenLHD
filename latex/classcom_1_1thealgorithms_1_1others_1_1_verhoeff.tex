\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classcom_1_1thealgorithms_1_1others_1_1_verhoeff}{Verhoeff}} algorithm is a checksum formula for error detection developed by the Dutch mathematician Jacobus \mbox{\hyperlink{classcom_1_1thealgorithms_1_1others_1_1_verhoeff}{Verhoeff}} and was first published in 1969. It was the first decimal check digit algorithm which detects all single-\/digit errors, and all transposition errors involving two adjacent digits.

The strengths of the algorithm are that it detects all transliteration and transposition errors, and additionally most twin, twin jump, jump transposition and phonetic errors. The main weakness of the \mbox{\hyperlink{classcom_1_1thealgorithms_1_1others_1_1_verhoeff}{Verhoeff}} algorithm is its complexity. The calculations required cannot easily be expressed as a formula. For easy calculation three tables are required\+:


\begin{DoxyEnumerate}
\item multiplication table 
\item inverse table 
\item permutation table 
\end{DoxyEnumerate}

\doxysubsection{Detailed Description}
This class implements a Cursor Linked List.

A \mbox{\hyperlink{classcom_1_1thealgorithms_1_1datastructures_1_1lists_1_1_cursor_linked_list}{Cursor\+Linked\+List}} is an array version of a Linked List. Essentially you have an array of list nodes but instead of each node containing a pointer to the next item in the linked list, each node element in the array contains the index for the next node element.

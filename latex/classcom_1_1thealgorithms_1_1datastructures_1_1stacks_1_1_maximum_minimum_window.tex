\doxysubsection{Detailed Description}
Given an integer array. The task is to find the maximum of the minimum of every window size in the array. Note\+: Window size varies from 1 to the size of the Array. 

For example, 

N = 7 arr\mbox{[}\mbox{]} = \{10,20,30,50,10,70,30\} 

So the answer for the above would be \+: 70 30 20 10 10 10 10 

We need to consider window sizes from 1 to length of array in each iteration. So in the iteration 1 the windows would be \mbox{[}10\mbox{]}, \mbox{[}20\mbox{]}, \mbox{[}30\mbox{]}, \mbox{[}50\mbox{]}, \mbox{[}10\mbox{]}, \mbox{[}70\mbox{]}, \mbox{[}30\mbox{]}. Now we need to check the minimum value in each window. Since the window size is 1 here the minimum element would be the number itself. Now the maximum out of these is the result in iteration 1. In the second iteration we need to consider window size 2, so there would be \mbox{[}10,20\mbox{]}, \mbox{[}20,30\mbox{]}, \mbox{[}30,50\mbox{]}, \mbox{[}50,10\mbox{]}, \mbox{[}10,70\mbox{]}, \mbox{[}70,30\mbox{]}. Now the minimum of each window size would be \mbox{[}10,20,30,10,10\mbox{]} and the maximum out of these is 30. Similarly we solve for other window sizes.

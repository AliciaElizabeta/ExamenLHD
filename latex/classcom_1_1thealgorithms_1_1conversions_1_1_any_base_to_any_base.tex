\doxysubsection{Detailed Description}
Class for converting from \char`\"{}any\char`\"{} base to \char`\"{}any\char`\"{} other base, when \char`\"{}any\char`\"{} means from 2-\/36. Works by going from base 1 to decimal to base 2. Includes auxiliary method for determining whether a number is valid for a given base.

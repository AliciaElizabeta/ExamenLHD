\doxysubsection{Detailed Description}
Least recently used (LRU) 

Discards the least recently used items first. This algorithm requires keeping track of what was used when, which is expensive if one wants to make sure the algorithm always discards the least recently used item. \href{https://en.wikipedia.org/wiki/Cache_replacement_policies\#Least_recently_used_(LRU)}{\texttt{ https\+://en.\+wikipedia.\+org/wiki/\+Cache\+\_\+replacement\+\_\+policies\#\+Least\+\_\+recently\+\_\+used\+\_\+(\+LRU)}}

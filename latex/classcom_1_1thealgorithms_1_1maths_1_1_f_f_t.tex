Iterative In-\/\+Place Radix-\/2 Cooley-\/\+Tukey Fast Fourier Transform Algorithm with Bit-\/\+Reversal. The size of the input signal must be a power of 2. If it isn\textquotesingle{}t then it is padded with zeros and the output \mbox{\hyperlink{classcom_1_1thealgorithms_1_1maths_1_1_f_f_t}{FFT}} will be bigger than the input signal.
